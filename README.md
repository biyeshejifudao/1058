# 网上蛋糕销售系统

## 项目简介
网上蛋糕销售系统是基于Java Web技术，Servlet，JSP页面技术、Mysql数据库的在线蛋糕销售系统。系统分为：前台系统（面向普通用户，在线售卖蛋糕），后台系统（对蛋糕产品、用户、类别、订单进行管理）。用户也分普通用户和管理员。 适合作为毕业设计、大作业、课程设计。
